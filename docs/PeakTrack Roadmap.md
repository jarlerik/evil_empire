
# 🗺️ PeakTrack – MVP Roadmap

## 🎯 Goal

Get real users to complete a full workout in PeakTrack, collect feedback, and decide on the next direction based on facts.

---

## 🟦 Phase 1 – MVP Finalization (Week 1) ✅

**Goal:** PeakTrack replaces the calculator + stopwatch + notes in a single workout.

**Tasks:**

- ✅ Finalize the workout view (clear flow)
- ⏱️ Add Rest Timer
    - Add rest time in sets & reps
    - Working / Resting
    - Sound
- 📝 Add set logging
    - Completed / failed
    - Weight + reps
- 💾 Supabase
- 📋 Simple History view
    - Last 30days


➡️ _At this stage, the app can be ugly — as long as it works._

---

## 🟦 Phase 2 – Internal Testing (Week 2)

**Goal:** Ensure a workout can be completed from start to finish without confusion.

**Tasks:**

- 🧪 Complete 3–5 full workouts yourself
- 🔍 Fix:
    - Slowness
    - Unclear buttons
    - Unnecessary clicks
- 🧹 UI cleanup
    - Font sizes
    - Contrast
    - Button placement

➡️ _Ask yourself: "Would I actually want to use this every workout?"_

## 🟦 Phase 3 – Soft Launch (Weeks 2–3)

**Goal:** 10–30 real users use PeakTrack in a real workout.

**Tasks:**

- 🚀 Release via:
    - TestFlight (iOS)
    - Play Store Internal Testing (Android)
- 👥 Recruit testers:
    - Gym buddies
    - Weightlifting groups
    - Coach contacts
- 📋 Feedback (5 questions):
    - What worked?
    - What was annoying?
    - What was missing?
    - Would you use it again?
    - Would you pay for this?

➡️ _More important than bugs: do users return for a second workout?_

## 🟦 Phase 4 – Iteration (Weeks 3–4)

**Goal:** Improve only what has proven value.

**Tasks:**

- 🧠 Analyze feedback
- ⭐ Choose 2 most important improvements
    - e.g. automatic weight rounding
    - faster set logging
- 🧼 Remove the unnecessary:
    - Features nobody uses
    - Extra views

➡️ _This is where the MVP starts to feel like a "real product."_

---

## 🟦 Phase 5 – Decision Point (Weeks 4–5)

**Ask yourself (honestly):**

- Are users completing more than one workout?
- Are they asking for more features?
- Is anyone already asking "when is this launching properly?"
- Are they willing to pay?

**Decisions:**

- 🟢 **GO** → add Supabase sync + onboarding
- 🟡 **ITERATE** → refine the core further
- 🔴 **STOP / PIVOT**
